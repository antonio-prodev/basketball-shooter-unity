# Basket Blitz

A 3D basketball shooting game built in Unity — 
an early project from my learning phase.

## Gameplay
Click and drag from the ball to aim, release to shoot. 
Score as many baskets as possible before time runs out. 
A new ball spawns after each shot using object pooling.

## Technical highlights
- Object pooling for ball management
- Physics-based projectile trajectory
- 3D environment with score and timer system
